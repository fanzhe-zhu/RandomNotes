RandomNotes settings

Summarize and Organize call the Anthropic Claude API directly from your browser. Your API key is stored locally in chrome.storage.local and is never sent anywhere except Anthropic.

Prompts

These are sent as the system field of the Claude API call. The user message is just your raw notes, unmodified. Edit to taste — leave blank to use the defaults.

Shortcuts

  • Ctrl/Cmd+Shift+S — add current selection to draft
  • Alt+Shift+N (or Option+Shift+N on macOS) — open the saved-notes popup
  • Or highlight text and click the floating + RandomNote button

Customize shortcuts at chrome://extensions/shortcuts.
